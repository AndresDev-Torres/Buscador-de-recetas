# 🍳 Buscador de recetas

App de búsqueda de recetas construida con React y la API de TheMealDB.

🔗 **Live demo:** https://buscador-de-recetas-weld.vercel.app/
## Features

- 🔍 Búsqueda de recetas en tiempo real
- ⌨️ Buscar presionando Enter o el botón Search
- 🍽 Vista de detalle con ingredientes e instrucciones
- 😕 Mensaje cuando no hay resultados
- 📱 Diseño responsivo

## Tech stack

- React
- Vite
- TheMealDB API
- CSS

## Run locally

1. Clona el repositorio
```bash
   git clone https://github.com/AndresDev-Torres/recipe-search.git
```
2. Instala las dependencias
```bash
   npm install
```
3. Inicia el servidor
```bash
   npm run dev
```
